The purpose of this exercise is to bring data from an OpenData link and create markers on OpenStreetMap's map through the use of Leaflet, AJAX and XML

In order to use it please open the file named 'AJAXJSON.html'
Clicking on the 'Load Test' button will create a marker at the 'Sainte Chapelle'
Clicking on the 'Load OpenData' button will create markers according to the data gathered from the link in the script of 'AJAXJSON.html'
