Ce script a pour but de permettre la connexion a un serveur qui renvois 3 etudiants au hasard parmis une liste.
Pour faciliter les tests chaque etudiants peut etre note jusqu'a trois fois.
Node-persist a ete choisi pour simplifier le systeme de persistance.
Pour visualiser cet exercice veuillez lancer Node.js.
Utilisez la commande 'npm install' pour installer les dependances.
utilisez la commande 'node app.js' pour lancer le script.
A l'aide de votre navigateur rendez vous sur 'localhost:3000'
